GetTempCompEnabled() As Boolean

Get whether or not software temperature compensation is used. To enable or disable software temperature compensation, call StartSingleSampleAcquisition or StartBufferedAcquistion with the desired setting.

Returns

True if software temperature compensation is available and enabled, false if temperature compensation is not available or not enabled. Also returns false if this calibration has not been initialized by calling LoadCalibrationFile
