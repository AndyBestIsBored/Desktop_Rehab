GetBiasSlope( index As Integer ) As Double

Get the bias slope values. The bias slope is used to perform software temperature compensation. For more information on the software temperature compensation algorithm, see Advanced Techniques.

Arguments

index

In: The index of the bias slope you wish to retrieve. Valid values are 0 to 5.

Returns

The bias slope at the specified index, or 0 if no calibration has been loaded using LoadCalibrationFile.
