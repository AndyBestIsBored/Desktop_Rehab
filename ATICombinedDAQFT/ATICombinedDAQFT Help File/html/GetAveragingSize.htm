GetAveragingSize() as Integer

Get the number of samples which are averaged together to reduce noise. Divide the 'raw' sampling frequency, as returned by GetSampleFrequency, by this number to get the effective sampling rate.

Returns

The number of raw samples which are averaged together to make one output sample.
