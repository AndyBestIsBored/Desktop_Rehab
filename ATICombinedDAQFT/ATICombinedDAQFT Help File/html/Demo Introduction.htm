Introduction to ATI DAQ F/T .NET Demo

Congratulations on your purchase of an ATI DAQ Force/Torque Transducer System. This application demonstrates some of the functionality of your system, and features a realtime display of forces and torques, full configuration options, and basic data logging capabilities.

This application uses the Microsoft .NET framework and the NI-DAQmx library and supports all National Instruments DAQ systems that can be used with the NI-DAQmx software version 7.3 or later. The hardware and drivers must be installed before running the demo.

The Visual Basic.NET source code for the demo and the .NET class library which is used to read the force and torque values is available from ATI Industrial Automation's website, www.ati-ia.com.

Features

One-shot Data Logging

Store a 6-axis measurement to a text file with the click of a button on the front panel. The time and date are automatically stored with the measurement. The resulting comma-delimited text file will open directly in Microsoft Excel and many other applications.

Continuous Data Logging

Store uninterrupted data from the DAQ F/T system to a file. This is perfect for situations where you need to know how the forces and torques change over time. The resulting comma-delimited text file will open directly in Microsoft Excel and many other applications.

Configure Your System

Once you have set up the output units, tool transforms, and other settings for your transducer, those settings are automatically saved for later use.

Software Filtering

Reduce the noise level in your measurements using a configurable averaging filter.
