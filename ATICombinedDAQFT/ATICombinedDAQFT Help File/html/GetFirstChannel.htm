GetFirstChannel() As Integer

Get the first (lowest-numbered) channel that the transducer is connected to. The ATICombinedDAQFT class library assumes that the transducer is attached to six or seven sequential channels of the same DAQ device, with gauge 0 connected to the lowest-numbered channel, gauge 1 connected to the next-loweset numbered channel, and so on. If software temperature compensation is used, the thermistor is expected to occupy the next lowest-numbered channel after gauge 5. If you bought your F/T system, including cables, from ATI Industrial Automation, the connectors are probably already in the correct order.

Returns

The first channel that the transducer is connected to.
